# Netflix Data Analysis Dashboard

This project analyzes Netflix content using Power BI.

## Tools Used
- Power BI
- Kaggle Dataset
- Data Visualization

## Key Insights
- Total Titles: 8808
- Genres: 36
- Directors: 4402
- Data from 2008 to 2021

## Dashboard Features
- Shows by rating
- Genre analysis
- Country distribution
- Movies vs TV Shows
- Release trend analysis
